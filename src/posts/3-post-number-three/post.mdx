<h1>JavaScript</h1> is relatively simply to begin as compared to other languageslike
C/C++, Java and Ruby. It doesn’t require any specific setup like installing IDE,
VM or hardware configuration – just a plain text editor and your web browser is enough.

<blockquote>
  JavaScript is the world's most misunderstood programming language. <br />
  Douglas Crockford
</blockquote>
